About

I am a programmer with a focus on machine learning and game development. My work involves creating neural networks for tasks like real-time hand tracking, reinforcement learning for games, and developing game engines from the ground up in C++.

I enjoy tackling complex challenges across the full project lifecycle, from data collection and model training to building custom physics and graphics engines.

Projects

Screenshot AI (SCAi)

AI that learns to play complex games from screen pixels.

Details
  • Concepts: CNNs, Autoencoders, Reinforcement Learning (PPO)
  • Trains a model to play Overwatch 2 by analyzing screenshots.
  • Includes separate neural networks for sensing in-game events like damage and health changes.
  • Addresses challenges of large input size with an image encoder.
AirHandWriter

Real-time hand tracking and pinch detection from a webcam.

Details
  • Concepts: Keypoint Detection, Gesture Recognition, CUDA
  • Uses a ResNet-based architecture to find thumb and index fingertips.
  • Features a custom PyTorch C++/CUDA extension for high-speed GPU data augmentation.
  • Includes a Flask web dashboard for managing experiments and hyperparameter sweeps.

A roguelike action game with a custom C++ 3D engine.

Details
  • Concepts: Game Engine Dev, 3D Graphics, Physics Simulation

  • Physics engine uses Verlet integration for character movement and cloth simulation.

RL Car

A car that learns to race around a track using Reinforcement Learning.

Details
  • Concepts: Reinforcement Learning, Simulation
  • The car uses 8 distance sensors as input to the model.
  • The track is a numpy array, allowing for easy collision detection.
  • Project includes a tool for creating new racetracks and a "human" mode for play.
Federated Learning

Distributed training of a text-generation RNN across multiple clients.

Details
  • Concepts: Distributed Systems, RNNs, NLP
  • Server-client architecture distributes model weights and aggregates loss values.

  • Uses a custom communication protocol for efficient data transfer.
Image Compression

Lossless image compression using a UNet and arithmetic coder.

Details
  • Concepts: Deep Learning, Data Compression, UNet, Attention
  • A UNet with self-attention predicts per-pixel probabilities.
  • A C++ arithmetic coder with Python bindings encodes the image to bits.
  • Pipeline supports variable-sized images without retraining the core model.
